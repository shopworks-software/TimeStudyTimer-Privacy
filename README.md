# Privacy Policy – Time Study Timer
Effective Date: 3/8/2026

Time Study Timer is developed and published by ShopWorks.

## Overview

Time Study Timer is a local-first productivity application designed for conducting cycle time studies and task timing in manufacturing and operations environments. The app does not collect, transmit, or store personal data on external servers.

All study data, notes, and exported files remain on the user's device unless the user chooses to manually share or export them.

## Data Collection

Time Study Timer does not:

- Require user accounts
- Collect personal information
- Store user data in the cloud
- Sell or share user data
- Track user activity

All time study data, tasks, notes, and results are stored locally on the user’s device.

## Permissions

Time Study Timer may request the following permissions:

### Storage / File Access
Used only when the user chooses to export or share a generated Excel report (.xlsx).  
This allows the app to save the export file to a location selected by the user.

The app does not access files outside of this user-initiated export process.

### Network Access
Used only when the user chooses to share an exported Excel file using Android’s system share menu (for example email, cloud storage, or messaging apps).

No data is transmitted to ShopWorks or any external servers.

## Third-Party Services

Time Study Timer does not use:

- Analytics services
- Advertising SDKs
- Tracking tools
- External cloud storage services

The application operates entirely on-device.

## Data Security

All study data is stored locally on the user’s device using the device’s standard application storage system. The app does not operate backend servers and does not transmit user data.

Users maintain full control over their data and exported files.

## Changes to This Policy

This Privacy Policy may be updated as the application evolves. Any changes will be reflected on this page with an updated effective date.
